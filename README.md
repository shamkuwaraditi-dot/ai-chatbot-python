# AI Chatbot (Python)

A simple rule-based AI chatbot made using Python.  
Built to demonstrate basic NLP logic and text matching.

## Features
- Greets the user
- Responds to simple phrases
- Rule-based message detection
- Randomized responses
- Easy for beginners to expand

## How to Run
1. Install Python 3
2. Download or clone the repository
3. Run in terminal:
 
python main.py

## Example Output
You: hi  
Bot: Hello!

You: how are you  
Bot: I'm doing great!

You: bye  
Bot: Goodbye!

## Project Files
- `main.py` — main chatbot logic  
- `requirements.txt` — dependencies  
- `README.md` — documentation
